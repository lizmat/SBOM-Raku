NAME
====

SBOM::Raku - Raku specific SBOM functionality 

SYNOPSIS
========

```raku
use SBOM::Raku;
```

DESCRIPTION
===========

The `SBOM::Raku` distribution provides logic for the creation and maintenance of SBOM (Software Bill Of Materials) files in the Raku Programming Language context.
